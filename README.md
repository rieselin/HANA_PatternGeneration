# HANAPatternGeneration

## Setup

```
>python3 -m venv ./venv/
# aktivieren dieser Umgebung
> .\venv\Scripts\activate
# Installation der wichtigsten Module
Module installieren
>pip3 install scipy matplotlib sympy jupyter
# Installation ngsolve
>pip3 install ngsolve
# Installation webgui für jupyter-notebooks
>pip3 install webgui_jupyter_widgets
webgui aktivieren.
# Aktivierung der Erweiterung in der jupyter-notebook Umgebung
>jupyter labextension enable webgui_jupyter_widgets
Jupytext für Markdown Files installieren und aktivieren
# Um direkt Markdown Files in Jupyter ausführen zu können benötigen wir noch:
>pip install jupyterlab-myst
>jupyter labextension enable jupyterlab-myst:executor
>pip3 install jupytext
>jupyter labextension enable jupytext

```
